# New Years Countdown
A basic countdown script I made in Python 3.

Tested and written entirely on the 30th December 2019.

## Requirements
* Python 3 (Tested to work on Python 3.8)
* pyttsx3
* A brain

## Installation and Usage
```
pip install pyttsx3
py NewYears.py
```
